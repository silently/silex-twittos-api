Twittos API Documentation

Introduction

The API root (noted below {api}) is twittos.grinos.com/api, meaning an API endpoint URI looks like http://twittos.grinos.com/api/tweets

API endpoints are meant to be called with parameters sent in a JSON body, and they respond with JSON descriptions too.

Users

Register

POST request to {api}/users with a JSON description containing:

  • login (mandatory)
  • password (mandatory with at least 6 characters)
  • email (mandatory)

Responses:

  • 201 (register ok, side-effect: creates a session and sets a cookie)
  • 409 (if login is already used)
  • 422 (with JSON data explaining error)

Get user profile

GET request to {api}/users/self

Responses:

  • 200 with the following fields:
    • login
    • email
  • 401 (not authenticated)

Authentication

Login request

POST request to {api}/sessions with a JSON description containing:

  • login (mandatory)
  • password (mandatory)

Responses:

  • 201 (login ok, sets a cookie)
  • 401 (failed: bad login/password, login does not exists...)

Logout request

DELETE request to {api}/sessions/self

Responses:

  • 200 (unsets cookie if any)

Tweets

List tweets

GET request to {api}/tweets, with an optional page query parameter (for example {api}/tweets?page=3) if you want to skip results.

Responses:

  • 200, array of 10 most recent (if page is 0 or not defined) tweets with the following fields for each tweet:
    • id
    • URI (to get more information)
    • text
    • authorLogin
    • authorURI (to get the author's tweets)
    • likes (number)
    • retweets (number)
    • createdAt
  • 401 (requires authentication)

Get more info on a given tweet

GET request to {api}/tweets/{id} where:

  • {id} is the tweet's id

Responses:

  • 200 with the following fields:
    • id
    • URI (current URI called by client)
    • text
    • authorLogin
    • authorURI (to get the author's tweets)
    • createdAt
    • isRetweet (boolean)
    • likes (number, 0 if isRetweet)
    • retweets (number, 0 if isRetweet)
    • orignalTweetId (present if isRetweet)
    • orignalTweetURI (present if isRetweet)
    • orignalTweetLikes (present if isRetweet)
    • orignalTweetRetweets (present if isRetweet)
    • orignalTweetAuthorLogin (present if isRetweet)
  • 401 (requires authentication)

Create tweet

POST request to {api}/tweets with a JSON description containing:

  • text (mandatory with a maximum of 140 characters)

Responses:

  • 201 (tweet created)
  • 401 (requires authentication)
  • 422 (bad parameters)

Delete tweet

DELETE request to {api}/tweets/{id} where:

  • {id} is the tweet's id

Responses:

  • 200 (tweet deleted)
  • 401 (requires authentication)
  • 403 (forbidden, for example if authenticated user is not the tweet author)

Like tweet

POST request to {api}/tweets/like/{id} where:

  • {id} is the tweet's id

Responses:

  • 201 (tweet like created)
  • 401 (requires authentication)
  • 404 (tweet not found)
  • 409 (already liked)

Retweet

POST request to {api}/tweets/retweet/{id} where:

  • {id} is the tweet's id

Responses:

  • 201 (retweet created)
  • 401 (requires authentication)
  • 404 (tweet not found)
  • 409 (already retweeted or author of the tweet)
